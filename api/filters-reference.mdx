`Filters` is a static factory class. Every method returns a `Filter` — a functional predicate over a `RequestContext`. Filters can be composed with `.and()`, `.or()`, and `.negate()`.

```java
Filter f = Filters.hasMessage().and(Filters.hasText());
Filter g = Filters.isPrivateChat().or(Filters.isGroupChat());
Filter h = Filters.isCommand().negate();
```

## Text matching

All text-matching filters operate on the full update text as resolved by `TextFinderService` (same as `As.text()`).

| Method | Matches when… |
|---|---|
| `textStartWith(String prefix)` | Text starts with `prefix` (case-sensitive). |
| `textEndsWith(String suffix)` | Text ends with `suffix` (case-sensitive). |
| `textContains(String substring)` | Text contains `substring` (case-sensitive). |
| `textContainsIgnoreCase(String substring)` | Text contains `substring` (case-insensitive). |
| `textEquals(String value)` | Text equals `value` (case-sensitive). |
| `textEqualsIgnoreCase(String value)` | Text equals `value` (case-insensitive). |

## Combinators

Use these when you need to apply the same filter-factory function to multiple values.

| Method | Description |
|---|---|
| `iterateOr(Function<T,Filter> factory, T first, T... rest)` | Returns `factory(first).or(factory(v1)).or(factory(v2))…` |
| `iterateAnd(Function<T,Filter> factory, T first, T... rest)` | Returns `factory(first).and(factory(v1)).and(factory(v2))…` |

```java
// Match any of several user IDs
Filter isAllowedUser = Filters.iterateOr(Filters::userIdIs, 111L, 222L, 333L);

// Message must contain all of these words
Filter hasAllKeywords = Filters.iterateAnd(Filters::textContainsIgnoreCase, "order", "confirm");
```

## Composition (instance methods on `Filter`)

Every `Filter` exposes three composition methods:

| Method | Description |
|---|---|
| `filter.and(Filter other)` | Both filters must match. |
| `filter.or(Filter other)` | Either filter must match. |
| `filter.negate()` | The filter must not match. |

```java
Filter privateTextMessage = Filters.isPrivateChat()
    .and(Filters.hasMessageText())
    .and(Filters.isCommand().negate());
```
